Bu eklenti Secretcv.com sitesinden belirlenen ayarlara göre iş ilanlarını alıp sitede yayınlar.

== Description ==

Secretcv.com sitesinde yayınlanan iş ilanlarından istediğiniz Sektör/Bölüm/Pozisyon/Şehir kombinasyonuna uyan ilanları
sitede gösterir.

== Installation ==

*   .zip arşivini açın.
*	/secretcv-is-ilanlari/ klasörünü 'wp-content/plugins' altına yükleyin.
*	Wordpress yönetim alanındaki "Appearance/Widgets" bölümünden aygıtı (widget) yönetebilirsiniz.
